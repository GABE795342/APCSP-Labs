Part I: What are vectors in p5?

  • Vector objects have only two properties: x and y

  • But they also have lots of methods that we can use . Here are some examples:

    set()   copy()  add()   sub()   mult()  div()  mag()  magSq()  
dot()  cross()  dist()  normalize() limit() setMag()  heading()  
rotate()  angleBetween()    lerp()  array() equals() fromAngle()  
fromAngles()    random2D()

  • Create a vector

    var location = createVector(200, 300);

This line of code creates a vector object where x = 200 and y = 300

    var velocity = createVector(random(-3, 3), random(-3, 3));

This line of code creates a vector object where x and y are random numbers between -3 and 3

function loadBalls(numBalls){

    for(var i = 0; i < numBalls; i++){
        var loc = createVector(random(width), random(height));
        var vel = createVector(random(-3,3), random(-3,3));
        var radius = random(20,40);
        var col = color(random(255), random(255), random(255));

        balls.push(new Ball(loc, vel, radius, col));

    }

}
